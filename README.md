### Требования

**[Node.js](https://nodejs.org/ "Node.js")**

### Установка

`npm install`

### Запуск

`npm start`

### Сборка

`npm run make`


### Список инструментов

Чтобы добавить список инструментов нужно указать <Брокер>:<Инструмент>

Например:

    BINANCE:BTCUSDT
    BINANCE:ADAUSDT
    BINANCE:ETHUSDT
